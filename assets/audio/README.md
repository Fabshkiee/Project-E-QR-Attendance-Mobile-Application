# Audio Assets

This directory should contain audio files for the QR scanner.

## Required Files:

- `success.mp3` - Success sound played when QR scan is successful

## Instructions:

1. Add your success sound file as `success.mp3` in this directory
2. Recommended format: MP3, 3-5 seconds duration
3. Keep file size small for better performance (< 500KB)

## Example Sources:

You can find free success sounds from:
- Freesound.org
- Zapsplat.com
- Pixabay audio
